Java 虚拟机（JVM）是 Java 程序运行的基础，理解 JVM 的内存模型和垃圾回收机制对于编写高性能、低内存泄漏的 Java 应用至关重要。本文将深入探讨 JVM 的内存结构、垃圾回收算法以及常见的 GC 调优策略。

  • Survivor 区：存放经过 Minor GC 后存活的对象
